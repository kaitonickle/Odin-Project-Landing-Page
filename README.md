# Flowstate — Landing Page

A productivity app landing page built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

The Odin assignment provides a generic mockup, but explicitly encourages substituting your own business, content, colors, and images. This project personalizes that structure into **Flowstate**, a fictional focus & productivity tool.

## Structure (matches assignment requirements)

1. **Header & hero** — navigation, headline, CTA, and product preview
2. **Features** — four feature cards with icons and descriptions
3. **Quote** — customer testimonial
4. **Call to action** — sign-up banner
5. **Footer** — copyright

Built with semantic HTML, a single `styles.css` file, and flexbox throughout.
